Every HTML element that holds any type of content can be a flexbox or grid container. 

The primary difference is that an id attribute must be unique to a single element on a page, while a class attribute can be reused on multiple elements.

In web development, we use the justify-content CSS property to control how the browser distributes extra space between and around items inside a flex or grid container ( MORE LIKELY TO CONTROL X AXIS )

While justify-content manages the "main" axis, we use align properties to control the opposite direction, known as the cross axis.(MORE LIKELY TO CONTROL Y AXIS)
